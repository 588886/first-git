git을 이용한 명령어(로컬)
touch 이름.파일형식(html등등) : 폴더 만들기 git version : git 버전확인
git config --list:깃 설정 확인
git config --global user.name : 이름 설정
git config --global user.email : 이메일 설정
git init : git 사용선언
git add index.html : index.html 깃 임시 업로드
git add . : 여러개 한번에 업로드 git commit -m "index.html 생성" : 깃 업로드
git status : 깃 현재 상태 확인
git rm --cached index.html : 깃 다시 원래대로
